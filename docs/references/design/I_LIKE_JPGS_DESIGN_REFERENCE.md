# I Like JPGs — Design Reference

This folder contains an HTML concept generated in Claude Design.

Use it as visual reference only.

Do not directly paste or port the full HTML into the Next.js app.

Extract:
- overall mood
- spacing
- dark/light balance
- card treatment
- typography rhythm
- button/input style
- section hierarchy
- playful but restrained visual language

Avoid:
- copying embedded fonts
- copying base64 assets
- copying generated inline CSS wholesale
- bringing in unnecessary animation or layout complexity

Milestone 0 should remain lightweight:
- landing/concept page
- wallet interest/input CTA
- manual taste report example cards
- interview-backed validation framing

The design should feel:
minimal, editorial, slightly playful, quietly premium, and clearly I Like JPGs.
