# I Like JPGs — Manual Taste Report Template

## Purpose

This template is for creating manual wallet taste reports during Milestone 0.

Manual reports help validate the product before the full classifier is perfect.

The goal is to see whether a collector recognizes themselves in the read and whether the structure feels useful, believable, and worth sharing.

## One-Line Read

A short, plain-English summary of what this wallet seems drawn toward.

Example:

> This wallet leans toward meme-native culture and social identity objects, with a secondary pull toward artist-led editions.

## Short Taste Read

Write 2–4 sentences.

The read should feel interpretive, not definitive.

Use language like:

- “This wallet seems drawn toward…”
- “There is a visible pattern of…”
- “The strongest signal appears to be…”
- “A secondary layer shows…”

Avoid:

- “This person is…”
- “This wallet proves…”
- “This collector ranks…”
- “This wallet is worth…”

## Object / Format Signals

Describe how the wallet collects, not just what it collects.

Examples:

- Edition-heavy
- ERC-1155-heavy
- PFP-heavy
- 1/1-like
- Access-pass-heavy
- Meme-card-heavy
- Interactive / animated
- On-chain
- Domain / identity objects
- World objects

## What This Wallet Seems To Recognize

Write 1 short paragraph.

This is the most human part of the report.

Example:

> This wallet seems to recognize NFTs as shared cultural language: cards, memes, avatars, editions, and references that signal participation in a particular internet-native world.

## What To Be Careful Not To Overclaim

List uncertainties.

Examples:

- Some metadata is thin.
- A few objects may be spam or airdrops.
- Some categories overlap.
- The wallet may contain transferred or gifted NFTs.
- This is a read of visible collecting patterns, not a complete identity.

## Suggested Follow-Up

What would be interesting to compare next?

Examples:

- Compare with another collector
- Compare with an artist wallet
- Find wallets with similar meme/art overlap
- Explore shared collections
- Look for communities this wallet might belong to
